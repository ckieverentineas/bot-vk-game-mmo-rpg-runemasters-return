# Changelog

Этот файл ведётся как пользовательский журнал изменений и не дублирует технические commit message.

## Политика релизов

- публичная версия считается по commit-based правилу формата `M.nn`;
- каждые `100` коммитов дают новый релизный рубеж;
- значит, `100` коммитов = версия `1.00`, `245` коммитов = версия `2.45`;
- текущее состояние можно посмотреть через `npm run release:status`.

## [Unreleased]

### Added

- единый каталог VK-команд для payload, алиасов и динамических действий;
- builder клавиатур, чтобы новые меню и кнопки добавлялись без копипасты;
- нулевой интро-биом с обучением, возможностью пропуска и возврата;
- умная динамическая сложность, учитывающая уровень, статы, победы и поражения;
- команда и кнопка `удалить персонажа`, чтобы не лазить вручную в базу во время тестов;
- release tooling для вычисления версии по количеству коммитов;
- unit tests для нормализации команд и релизной формулы.

### Changed

- ручная смена уровня локации отключена, теперь давление мира подбирается автоматически;
- VK-сценарий доведён до keyboard-first формата: основные действия доступны кнопками;
- зависшие активные бои теперь автоматически восстанавливаются и корректно завершаются;
- боевой движок и движок наград переведены на общие combat utilities;
- лимит журнала боя теперь берётся из `gameBalance`, а не хранится магическим числом;
- Prisma-репозиторий использует общие JSON helper'ы и единый mapper battle persistence;
- `GameHandler` разбит на маленькие reply/helper-методы, чтобы новые команды добавлялись без разрастания switch-блока;
- `Logger` стал типобезопасным и больше не использует `any`.

### Docs

- переписаны `README.md`, `QUICKSTART.md` и `ARCHITECTURE.md`;
- changelog переведён с рекламного описания на поддерживаемый рабочий формат.
